Welcome to the Human Activity Recognition System

This project uses advanced sensor data and machine learning models to accurately predict human activities such as walking, sitting, standing, and more. Built using Python, Flask, and a Logistic Regression model, this system processes real-time or static input from accelerometers and gyroscopes to classify activities.

About This Project

The Human Activity Recognition (HAR) system leverages wearable sensor data to classify six core activities: WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, and LYING. The data is processed using StandardScaler and predictions are made with a trained Logistic Regression model. This can be used in healthcare monitoring, fitness tracking, and smart environments.
